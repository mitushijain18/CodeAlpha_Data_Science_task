# Car Price Prediction Model

## 📝 Overview

This project uses Machine Learning to predict the market price of used cars based on various features such as brand, model, year of manufacture, fuel type, transmission, and kilometers driven. The goal is to provide an accurate estimate of a vehicle's value using data-driven insights.

## 🚀 Pipeline Steps

* **Data Preprocessing:** Cleaning raw data, handling missing values, and encoding categorical variables.
* **Exploratory Data Analysis (EDA):** Visualizing relationships between car features and their impact on price.
* **Feature Engineering:** Creating new features to improve prediction accuracy.
* **Model Selection:** Comparing multiple algorithms (e.g., Linear Regression, Random Forest, XGBoost) to find the best fit.
* **Evaluation:** Measuring performance using metrics like R-squared and Mean Absolute Error (MAE).

## 🛠️ Technologies Used

* **Languages:** Python
* **Libraries:** `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`, `Scikit-learn`
* **Tools:** Jupyter Notebook / VS Code
